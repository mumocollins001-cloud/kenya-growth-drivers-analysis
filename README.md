# Kenya Growth Drivers Analysis

## Project Overview

This project investigates the relationship between key macroeconomic indicators and Kenya's GDP Growth using World Bank data.

Variables analyzed:

* GDP Growth
* Investment (% of GDP)
* Inflation (%)
* Government Expenditure (% of GDP)
* Unemployment (%)

## Objectives

* Identify variables associated with GDP Growth.
* Apply statistical analysis techniques.
* Build and interpret regression models.
* Communicate findings through visualizations.

## Data Source

World Bank - World Development Indicators (WDI)

## Tools Used

* Python
* Pandas
* Matplotlib
* Statsmodels
* PowerShell
* GitHub

## Methods

* Data Cleaning
* Data Transformation
* Correlation Analysis
* OLS Regression
* Hypothesis Testing
* Multicollinearity Testing (VIF)
* Visualization

## Key Findings

### Significant Variables

* Investment (+)
* Inflation (-)

### Not Statistically Significant

* Government Expenditure
* Unemployment

### Final Model

GDP Growth = f(Investment, Inflation)

R² = 0.319
